R

R software must be installed.

Rstudio

We highly recommend that you have installed R-studio.

gen3sis R-package and dependencies

You can find information on how to install gen3sis in this introductory vignette.

It would be advantegeous (although not necessary) if you already start familiarizing yourself on how to create gen3sis landscapes and configurations.

If you already have a an existing landscape that you would like to use, please bing a gen3sis landscape object already compiled. Follow this vignette for more information. Note that landscapes with more than ~ 100 timesteps and ~2000 sites might be time consuming. We recommend that you subsample or coarsen the spatial and temporal resolution of the landscape that you will use during the breakout rooms.
